## OZ Event Services — Official Website

Premium, conversion-focused company website built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Lucide icons.

### Tech stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

### Local development

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

### Production build

```bash
npm run lint
npm run build
npm run start
```
